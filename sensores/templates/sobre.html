Sobre o AgroStation

Sistema completo de monitoramento ambiental para aplicações agrícolas

Visão Geral

O AgroStation é uma plataforma avançada de monitoramento ambiental desenvolvida especificamente para aplicações agrícolas. Utilizando tecnologias de IoT (Internet das Coisas), o sistema fornece dados em tempo real sobre condições ambientais, qualidade do ar, segurança e parâmetros meteorológicos.

Tecnologias Utilizadas

Backend

  • Django Framework
  • Django Channels (WebSockets)
  • PostgreSQL
  • Redis

Comunicação

  • MQTT Protocol
  • RabbitMQ Broker
  • WebSocket Real-time

Frontend

  • HTML5 & CSS3
  • JavaScript ES6+
  • Chart.js
  • Leaflet Maps

Sensores Integrados

Temperatura

DHT22 e BMP280

Umidade

Ar e Solo

Luminosidade

Sensor de luz

Vento

Velocidade e direção

Chuva

Detecção de precipitação

Qualidade do Ar

PM1.0, PM2.5, PM10

Gás

MQ-2 (vazamentos)

GPS

Localização geográfica

Funcionalidades Principais

  • Dashboard em Tempo Real: Visualização instantânea de todos os sensores com atualizações via WebSocket
  • Análise Histórica: Estatísticas detalhadas por período (hoje, semana, mês)
  • Geolocalização: Mapa interativo com localização precisa da estação
  • Alertas Inteligentes: Notificações para valores críticos e situações de risco
  • Interface Responsiva: Acessível em qualquer dispositivo (desktop, tablet, mobile)
